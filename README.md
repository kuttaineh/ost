### Terminal based speech tool

* Click [ Code ] Download ZIP
* Click ost-main.zip
* Open Terminal, copy/paste the following:
```
cat>>.zshrc
```
Press [return]
```
alias ost='~/Downloads/ost-main/ost'
alias ps='~/Downloads/ost-main/ps'
```
Press [return]
Press [control][D]

#### To run
```
ost
```

#### To exit
```
[control][C]
```
