### Terminal based speech tool

* Click [ Code ] Download ZIP
* Click ost-main.zip
* Open Terminal, copy/paste the following:
```
cat>>.zshrc
```
Press [return]
```
alias ost='~/Downloads/ost-main/ost'
alias pps='~/Downloads/ost-main/pps'
```
Press [return]
Press [control][D]
```
source ~/.zshrc
```
Installation is now complete.
#### To run
```
ost
```

#### To exit
```
[control][C]
```
